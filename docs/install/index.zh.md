# 安装 Pinferencia

## 推荐方式

推荐将 `Pinferencia` 与 `uvicorn` 同时安装，这样可以立刻启动你的服务.

<div class="termy">

```console
$ pip install "pinferencia[uvicorn]"
---> 100%
```

</div>

### 或者...

你也可以选择仅 `Pinferencia`。稍后你可以再选择其它的ASGI Server。

<div class="termy">

```console
$ pip install pinferencia
---> 100%
```

</div>
